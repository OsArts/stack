# Docker

## Primary commands

- `$ docker ps`
- `$ docker info` :: `// [swap limits]` [docker warning no swap limit support](https://qastack.ru/unix/342735/docker-warning-no-swap-limit-support)
- `$ docker images` - return list images saved into local machine.

## Ubibntu

- `$ uname -or` || `$ uname -a`
- `$ cat /proc/version`
- `$ cat /etc/lsb-release`
